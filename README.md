
# CryptoCharts

This is a reconstruction of a project (original source code unavailable) that I did in early 2019 as part of my interview as a technical content writer at FusionCharts.

Features:
- BTC/USD, BTC/EUR, ETH/USD, ETH/EUR dashboard
- Currency converter
- FusionCharts real-time chart
- CoinGecko API integration
- Pure HTML/CSS/JavaScript

Open `index.html` in a browser or serve via a local web server.
